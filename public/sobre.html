Sobre o Projeto

Olá! Meu nome é Lucas Bruno e este é o projeto Cupcakes do Lucas, desenvolvido como parte dos meus estudos em Engenharia de Software.

O objetivo deste site é criar uma vitrine digital de cupcakes artesanais, com um layout moderno, responsivo e funcional. Aqui você pode explorar nossas categorias de produtos, ver imagens, descrições e adicionar itens ao carrinho de forma interativa.

Este projeto foi desenvolvido com:

  • HTML5 para a estrutura das páginas
  • CSS3 para o design e responsividade
  • JavaScript para funcionalidades dinâmicas

O desenvolvimento foi feito na IDE Visual Studio Code, com controle de versão pelo GitHub. O site está hospedado online através do Firebase Hosting, permitindo acesso rápido e seguro.

Obrigado pela visita ;) espero sua avaliação.
