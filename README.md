# Projeto-PIT-2025-2
--Projeto PIT 2025 Cupcakes do Lucas--

Bem-vindo(a) ao Cupcakes do Lucas, meu projeto desenvolvido com muito carinho para apresentar uma vitrine digital de cupcakes artesanais.
O objetivo deste site é proporcionar uma experiência agradável, simples e funcional para a avaliação do meu projeto PIT.


--Sobre o Projeto--

Este projeto foi desenvolvido com o intuito de aplicar e demonstrar meus conhecimentos em desenvolvimento, gerenciamento de versões e publicação.
A ideia foi criar uma página web completa, com design atrativo, layout responsivo e interatividade entre os elementos.

A aplicação simula uma pequena loja virtual onde é possível:

Visualizar cupcakes com imagens e descrições;

Adicionar produtos ao carrinho;

Ver o contador de itens atualizando dinamicamente.

O design foi pensado para ser limpo, moderno e intuitivo, destacando os produtos de forma agradável.



--Tecnologias Utilizadas--

Durante o desenvolvimento utilizei as seguintes tecnologias:

HTML: Estruturação da página e conteúdo

CSS: Estilização, cores, tipografia e responsividade

JavaScript: Lógica de interação com o usuário e contador do carrinho

Firebase Hosting: Hospedagem do site de forma simples e eficiente

Git e GitHub: Controle de versão e armazenamento do código

Visual Studio Code: IDE principal para desenvolvimento

Drawio: Criação do Wireframe 


--Desenvolvimento--

Desenvolvi todo o projeto utilizando o Visual Studio Code como ambiente de desenvolvimento.
Durante o processo, utilizei o GitHub para versionamento e backup do código, o que facilitou o gerenciamento das alterações e garantiu a segurança do trabalho.

Publiquei o site utilizando o Firebase Hosting, o que permite o acesso online e garante boa performance e estabilidade.
